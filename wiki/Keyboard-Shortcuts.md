# Keyboard Shortcuts

Press **`?`** anywhere in the app to open the shortcut help overlay.

---

## Global

| Key | Action |
|-----|--------|
| `?` | Open / close keyboard shortcut help |

---

## Tool Library — table navigation

| Key | Action |
|-----|--------|
| `/` | Focus the search box |
| `j` or `↓` | Move focus to the next row |
| `k` or `↑` | Move focus to the previous row |
| `Enter` or `e` | Open the editor for the focused row |
| `Space` | Toggle selection on the focused row |
| `Ctrl+A` | Select all visible (filtered) tools |
| `Esc` | Clear selection / close panel |

---

## Tool Library — toolbar actions

| Key | Action |
|-----|--------|
| `Ctrl+Q` | Open QR scanner (Maintain ▾ → Scan QR) |

---

## Tool editor

| Key | Action |
|-----|--------|
| `Ctrl+S` | Save the current tool |
| `Ctrl+Z` | Undo last change |
| `Ctrl+Shift+Z` | Redo |
| `Esc` | Close editor (discards unsaved changes) |

---

## Notes

- All shortcuts work when focus is on the table or on the page background, not inside a text input.
- The editor undo/redo history is **per session** — it resets when the editor is closed.
- `Ctrl+A` selects only the **visible** (filtered) tools, not the entire library.
